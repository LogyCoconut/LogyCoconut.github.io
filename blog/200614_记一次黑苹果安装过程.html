    # 关机后会自动重启的一个坑

    # 问题描述

    刚装的时候是没有这个问题的，但是在使用一段时间后发现macOS在关机后（风扇灯已经熄灭）会再次开机，关机流程变成了关机 -> 重启 -> 关机，这样才能真正关机

    # 解决办法

    关闭BIOS中的 SETTINGS \ 高级 \ 唤醒事件设置 \ USB设备从S3/S4/S5唤醒即可解决
