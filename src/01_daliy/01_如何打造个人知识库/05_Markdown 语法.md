### 引用式链接

之前是这么写链接的

```
这是一个 [链接](链接地址)
```

但是还有一种更加优雅的方式

```
这是一个 [链接id][]

[链接id](链接地址)
```

亦或者

```
这是一个 [链接别名][链接id]
[链接id](链接地址)
```
