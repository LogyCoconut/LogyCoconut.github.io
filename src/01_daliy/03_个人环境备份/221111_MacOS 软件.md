## Mac使用软件

> 以下软件都可以在官网下载，不要轻易相信三方网站

### 开发

| 名称                 | 介绍                   |
| :----------------- | :------------------- |
| IDEA CE            | 社区版的功能似乎够用           |
| Visual Studio Code | 部署博客时需要              |
| Sublime Text       | 记录一些文本、代码碎片时很有用，打开很快 |
| iTerm              | 终端工具                 |
| Termius            | 链接                   |
| Sequel Pro         | 免费MySQL客户端           |
