### 开发

| 软件名                | 备注                   |
| :----------------- | :------------------- |
| IDEA CE            | 社区版的功能似乎够用           |
| Visual Studio Code | 部署博客时需要              |
| Sublime Text       | 记录一些文本、代码碎片时很有用，打开很开 |
| iTerm              | 终端工具                 |
| Termius            | 链接                   |
| Sequel Pro         | 免费MySQL客户端           |

_另外： 「Sequel Pro」的正式版在 Mac 上打开有点问题，可以下载[测试版本](https://sequelpro.com/test-builds)。_
