# Sublime Text 4 配置文件

- `CMD + SHIFT + P`，安装 Package Control
- 安装 `Ayu` 主题
    _主题安装需要在 `Package Control: Install Package Package` 选项中搜索主题
- 粘贴配置文件 Json

```json
{
    "theme": "ayu-dark.sublime-theme",
    "font_size": 17,
    "update_check": false,
    "ignored_packages": [
        "Vintage",
    ],
    "color_scheme": "ayu-dark.sublime-color-scheme",
}
```
