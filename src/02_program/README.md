## 针对自己的 Java 学习一条龙

### Java 入门

- 语法基础
- Java 8 新特性（Stream、Lambda、新的时间 API）
- 数据结构与算法
- 计算机基础（网络、操作系统）

### Java 高级

- 多线程（JUC）
- JVM

### 企业级开发

- Web 开发（Servlet、Json、Cookie、Session）
- MySQL
- 框架
  - Spring（IOC、AOP、事务）
  - Spring MVC
  - Mybatis
  - SpringBoot
- 工具
  - Git
  - Linux
- 开发规范（阿里巴巴Java开发手册、Google Java Style）

### 进阶

- 设计模式
- 中间件（Redis、ES、MQ）
- Netty
- 微服务
- 架构设计（分布式、高可用、高并发）
