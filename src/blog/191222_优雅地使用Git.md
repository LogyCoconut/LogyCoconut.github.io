### gitmoji使用

> emoji不能随意的使用，在gitmoji官网中就有着不同emoji对应着的不同释义

#### 提交格式

`git commit -m ':emojicode: gitmoji演示'`

#### 常用的几个moji表情

| emoji |      emoji代码     |      说明     |
| :---: | :--------------: | :---------: |
|   🎨  |       `art`      | 改进代码结构/代码格式 |
|   🐛  |       `bug`      |    修复bug    |
|   ⚡️  |       `zap`      |     提升性能    |
|   🔥  |      `fire`      |   移除代码或文件   |
|   📝  |      `memo`      |     写文档     |
|   ✅   | `ite_check_mark` |     增加测试    |
|   🔒  |      `lock`      |    修复安全问题   |
|   💩  |     `hankey`     | 编写需要改进的错误代码 |
