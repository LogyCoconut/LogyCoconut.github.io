_主题安装需要在 `Package Control: Install Package Package` 选项中搜索主题

```json
{
    "theme": "ayu-light.sublime-theme",
    "font_size": 17,
    "update_check": false,
    "ignored_packages": [
        "Vintage",
    ],
    "color_scheme": "Packages/ayu/ayu-light.sublime-color-scheme",
}
```
